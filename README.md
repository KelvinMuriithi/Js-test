# Mini Jest Project

A simple mini project for writing and running JavaScript tests using **Jest**.

## Features

- Write unit tests in JavaScript
- Run tests with Jest
- Easy project structure for beginners

## Getting Started

### 1. Initialize the project

```bash
npm init -y
```

### 2. Install Jest

```bash
npm install --save-dev jest
```

### 3. Update `package.json`

Add this inside the `scripts` section:

```json
"scripts": {
  "test": "jest"
}
```



## Run Tests

```bash
npm test
```


## Notes

- Keep source files inside the `src` folder.
- Keep test files inside the `tests` folder.
- Name test files with `.test.js` for clarity.
